_options

_options: { delay: number; enabled: boolean; interval: number; queriesPerPeriod: number; timeout: number }

Type declaration

  • delay: number

    how long to wait before starting the first random walk, in milliseconds (default: 10000)

  • enabled: boolean

    discovery enabled (default: true)

  • interval: number

    how often to run the the random-walk process, in milliseconds (default: 300000)

  • queriesPerPeriod: number

    how many queries to run per period (default: 1)

  • timeout: number

    how long to wait for the the random-walk query to run, in milliseconds (default: 30000)

start

  • start(): void

  • Start the Random Walk process. This means running a number of queries every interval requesting random data. This is done to keep the dht healthy over time.

    Returns void

stop

  • stop(): void

  • Stop the random-walk process. Any active queries will be aborted.
